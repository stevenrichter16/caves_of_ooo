## 6. Anti-style self-check

closure for it. the* th. *is the body's only

*the body's only* — plain craft-clerical compounds,
lower-case, no archaism. *Closes*, *has*, *is* are
contemporary; no fronted nominal, no *hath /
lieth / unto / verily / betwixt / wherein*.
