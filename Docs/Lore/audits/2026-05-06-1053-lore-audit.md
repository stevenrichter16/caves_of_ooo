### 2.1 🟡 Hygiene 8 broadens criterion-1 silently

**File:** `2026-05-06-1002-lore-audit.md` §4 hygiene 8
proposal, all three options (a / b / c).

The 0957 §3.2 *Mode A criterion-1* reads in full:

> *the most-recent-sitting was Mode B and the chain's recent
> output has been mostly content (**multiple Mode B in a row**
> drift toward chain-extension by accretion).*

Two conjuncts: (a) most-recent Mode B; (b) **multiple Mode B
in a row.** The criterion fires only when both hold.

The 1002 §4 calls this same criterion *"alternation"* and
treats it as conditional on (a) alone. All three of the 1002's
hygiene-8 options ((a) conservative, (b) liberal, (c) hybrid)
inherit this implicit broadening — they trigger criterion-1
on *any* Mode B, not on multiple-in-a-row.

This is a substantive rule shift, not a clarification. Under
0957 §3.2 as written: a single prior Mode B does not invoke
criterion-1; the chain enters criterion-1 territory only at
the second consecutive Mode B. Under hygiene 8 as ratified at
1002 §4: a single prior Mode B *does* invoke alternation.

**Severity 🟡.** The 1002's hygiene-8 ruling stands as the
chain's working rule (the 1002 explicitly added it to the
roster), but the relationship to the 0957's original wording
is not honest. The 1053 surfaces the discrepancy without
overturning the rule — *the 1002's hygiene 8 silently amended
the 0957's §3.2, and the audit chain should accept the
amendment with explicit notice.*

**Recommendation.** A future Mode A or Mode B sitting (when
the override-quota refreshes at sitting 28) should issue an
explicit hygiene-8 amendment-note: *the 0957 §3.2 criterion-1
is hereby broadened to fire on a single prior Mode B; the
*multiple-in-a-row* clause is retained only as the
three-consecutive-Mode-B reset rule.* This separates the two
instruments cleanly.

### 2.2 🔵 The 1002 §4 *"criterion-1 fires"* analysis is true under hygiene 8 but false under 0957 §3.2 strict reading

**File:** `2026-05-06-1002-lore-audit.md` §4 *The disagreement

## 4. Anti-style self-check

**Quoted sentence (from §3.4):**

> *"Three consecutive Mode B is now unreachable — the quota
> allows at most one Mode B per four-window."*

Working register, gloss-form. *Quota* and *four-window* are
flat working-bureaucratic compounds the chain uses about its
own working-rule. *Unreachable* is the working logical
predicate (state-space term, not mythic). The sentence's
claim is structural about the rule's reachable states, not
moralised about virtue or vice. Lower-case throughout. No
allegorical capitalisation. The hyphen-stitched compounds
(*four-window*, *Mode B*) are working-style, not heraldic.

TES rewrite (forbidden): *"Three Modes of Bounty in a row are
become Forbidden, by the Holy Quota that grants but One Lapse
in every Tetrad of Sittings."* Tells: capitalisations on
*Modes*, *Bounty*, *Forbidden*, *Holy Quota*, *Lapse*,
*Tetrad*, *Sittings*; *are become* (KJV-archaic copular);
*Tetrad* mythologising the working *four-window*; *Lapse*
moralising the working *override*. The actual sentence:
*three*, *consecutive*, *Mode B*, *unreachable*, *quota*,
*four-window* — flat working terms, the structural claim
doing its own work.

Pass.
