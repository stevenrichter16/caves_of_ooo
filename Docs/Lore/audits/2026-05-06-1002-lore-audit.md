## 4. Contradictions and bleeds — cold-eye on 0957's two-mode hold-discipline

The 0957's §3.2 listed five mode-switching criteria, three
under *Choose Mode A* and two under *Choose Mode B*. **The
listing is parallel; the priority-ordering between the
criteria when they disagree is unstated.**

**The disagreement that exposes the gap.** Criterion-1 of
*Choose Mode A* says: *"the most-recent-sitting was Mode B
and the chain's recent output has been mostly content (multiple
Mode B in a row drift toward chain-extension by accretion)."*
Criterion-2 of *Choose Mode B* says: *"the most-recent-sitting
was Mode A and a genuinely additive deepening or resolution
edit IS available."* These two are **near-mirrors** with one
content-component (the *additive-available* clause) appearing
only on the Mode-B side.

When can they disagree? When the most-recent-sitting was Mode
B and there is additive content available. In that case:

  - Mode-A criterion-1 fires (recent-sitting was Mode B).
  - Mode-B criterion-2 does *not* fire (it requires the
    most-recent-sitting to be Mode A).

So at first glance the two criteria do *not* disagree in this
case. The 0957 is the first counter-example: 0915 was Mode B,
0957 had additive content available, 0957 chose Mode B
anyway. **The 0957 chose Mode B by an unstated criterion** —
neither criterion-1 of Mode A nor criterion-2 of Mode B
matched; the 0957 invented a new selection: *if the most-recent
sitting was Mode B but additive content is available, choose
Mode B again.*

This is a real choice and may be the right one. But the
two-mode discipline as written does not *capture* that
choice. The 0957 §3.3 notes only that it *had* Brine
deepening available; it does not derive Mode B from §3.2
explicitly.

**Severity 🟡.** Methodology-side. Without a precedence
rule, the next sitting that hits the same condition (most-
recent Mode B + additive content available) has to re-derive
the choice from scratch, and may not derive consistently with
the 0957's choice.

**Proposed fix (new hygiene item 8 — *criterion-priority
precedence*).** Either:

  - **(a) Conservative — alternation-first.** Mode-A
    criterion-1 (alternation) takes precedence over Mode-B
    criterion-2 (additive content). Result: most-recent Mode B
    + content available → **Mode A.** The 0957 should have
    been Mode A. Going forward, this sitting (1002) is
    correctly Mode A by the conservative read. Maximises
    discipline-reset frequency; risks under-shipping additive
    content the chain has cheap.

  - **(b) Liberal — additive-first.** Mode-B criterion-2
    (content-available) takes precedence over Mode-A
    criterion-1 (alternation). Result: most-recent Mode B +
    content available → **Mode B.** The 0957 was correctly
    Mode B. Going forward, the 1002 should be Mode B if
    additive content is available. Maximises content-shipping
    when cheap; risks Mode-B accretion (the very failure mode
    the §3.1 *three-consecutive-Mode-B discipline-reset* rule
    was designed to catch).

  - **(c) Hybrid — alternation-first with override-quota.**
    Default to (a). Allow at most one criterion-priority
    *override* per N sittings (suggested N = 4, matching the
    three-consecutive-Mode-B reset rule). The 0957's choice
    of Mode B counts as that override; 1002 must be Mode A.

**Recommendation.** Adopt **(c)** — hybrid. The 0957's
selection of Mode B was on-balance correct (the Brine
deepening was genuine, the keel-singer pedagogy was useful)
but the rule that allowed it should be explicit and quota-
limited. (c) gives the chain *both* alternation-discipline
*and* a documented escape valve, which is what 0957 used
implicitly. The escape-valve quota turns the implicit rule
into a working-grammar.

**File as item 42 — methodology-side, audit-internal hygiene

ambiguity in 0957's §3.2.

## 6. Anti-style self-check

**Quoted sentence (from §4 above):**

> *"Mode-A criterion-1 (alternation) takes precedence over
> Mode-B criterion-2 (additive content)."*

This is a working-rules sentence, not a TES-book sentence.
*Mode A* and *Mode B* are flat working-labels, not
allegorical capitalisations (cf. *the Sacred Mode*, *the
Lesser Path*). *Criterion-1* and *criterion-2* are bare
ordinals, the way a draftsman labels alternates. The verb
*takes precedence over* is bureaucratic-procedural, not
ritualistic. The sentence's claim is methodological — *if
two rules disagree, the alternation rule wins* — and reads
like a footnote in a process manual.

TES rewrite (forbidden): *"Verily, the First Discipline of
Alternation hath dominion over the Second Discipline of
Bounty, as the chronicler-priest of old hath ordained, lest
the Sittings drift into accretion and dishonour."* Tells:
*verily*; *hath* (twice); allegorical capitalisations on
*Discipline*, *Sittings*, *Bounty*; *chronicler-priest of old*
mythologisation; *dishonour* moralised consequence; pseudo-
medieval syntactic inversion. The actual sentence: lower-
case throughout, ordinals-as-labels, plain procedural verb,
zero mythologising appositives.

Pass.
